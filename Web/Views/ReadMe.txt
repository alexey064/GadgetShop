Здесь находятся html формы, ответственные за клиентскую часть магазина.
В папке Shared файлы, начинающиеся с нижнего подчеркивания используются другими
областями (areas). Файлы без подчеркивания используются в только в данной области.
Файлы без нижнего подчеркивания преимущественно используются в качестве шаблона 
объектов магазина.

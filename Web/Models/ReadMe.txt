В данном разделе находятся различные классы, используемые различными частями приложения.
Папка EF состоит из различных классов, используемых в качестве баз данных.
Папка Interfaces содержит некоторые интерфейсы, на данный момент папка почти не используется.
Папка Models содержит папки Linked и simple. В simple находятся независимые сущности,
в Linked - сущности, зависимые от сущностей в Simple.
